Deckbuilding with Untap

To start your first deck or a new one, go to untap.in and select the decks tab, from there you can create a new deck, or edit an existing deck. Make sure to select "MTG" and give your deck a name. The bar on the left will let you search for cards, or select paste deck from the hamburger menu to import a decklist.


You can also select download deck and follow those instructions which can be used to create a .txt file that you can copy and paste all of the cards into a comment on the document below to share your deck with others.
